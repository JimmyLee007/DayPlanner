# Day Planner 

Generates a random message in the form:

```
Todays morning excercise will be{form of excercise}. The main meal of the day is {meal}. This evening you will {activity}.
```

The technologies / programs used are
 - JavaScript
 - Node.js
 - VS Code
 - Markdown
 - Git
 - GitHub
