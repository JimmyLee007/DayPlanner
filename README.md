# Day Planner 

Generates a random message in the form:

```
Todays excercise session is {form of excercise}, main meal of the day is {meal} and to relax this evening you will {activity}.
```

The technologies / programs used are
 - JavaScript
 - Node.js
 - VS Code
 - Markdown
 - Git
 - GitHub
